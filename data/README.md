# Data

This repository does not include the raw input datasets used in the thesis.

Place local copies of the following files in this directory before running the scripts:

- `epidemiology.csv`
- `mobility.csv`
- `oxford-government-response.csv`
- `weather.csv`
- `RIVM_cases.csv`
- `RIVM_cases2.csv`

The new Python scripts read directly from the data directory you pass with `--data-dir`.

Large raw datasets are intentionally kept out of version control.
